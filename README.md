# Hosting Lesson Static

Маленький статический сайт для урока по хостингу.

## Что внутри

- `index.html` — структура страницы.
- `styles.css` — внешний вид страницы.
- `script.js` — маленькая проверка, что JavaScript работает.

## Как открыть локально

Открой файл `index.html` в браузере.

## Как опубликовать позже

1. Сохрани папку проекта в GitHub.
2. Подключи репозиторий к Vercel.
3. В Vercel оставь настройки как для статического сайта: build command пустой, output directory пустой или `.`.
